Question-1: What are the different ways to select an element in the DOM?

There are several ways to select an element in the DOM in JavaScript. Some of the ways are mentioned below: 1. Selecting using ID 2. Selecting using Class 3. Selecting using Tag Name 4. Selecting using CSS selectors querySelectorAll() (selects all matching elements)

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML= Can modify and read both HTML and text. innerText= Takes only visible text, ignoring HTML tags. textContent= Ignores HTML tags, can even take hidden text.

Question-3: What is event delegation in the DOM?

Event delegation is a technique for handling events on a parent element by adding event listeners to its child elements.

Question-4: What is event bubbling in the DOM?

Event Bubbling is a process where an event first occurs on the target element, then propagates stepwise to its parent, grandparent, etc., higher levels of the DOM.
